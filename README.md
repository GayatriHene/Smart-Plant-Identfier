# 🌱 Smart Plant Identifier

## 📌 Overview
Smart Plant Identifier is an AI-based application that identifies plant species using image input and machine learning techniques.

## 🚀 Features
- 🌿 Plant identification using images  
- 🤖 Machine Learning model integration  
- 📊 Data processing and analysis  

## 🛠 Tech Stack
- Python  
- Machine Learning  
- Computer Vision  

## ▶️ How to Run
1. Clone the repository  
   git clone https://github.com/GayatriHene/Smart-Plant-Identfier.git  

2. Install dependencies  
   pip install -r requirements.txt  

3. Run the project  
   python main.py
